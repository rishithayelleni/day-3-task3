This project focuses on performing Exploratory Data Analysis (EDA) on the **Netflix Movies and TV Shows** dataset.  
The goal is to understand data patterns, trends, and relationships using visualization techniques.

- **Name:** Netflix Movies and TV Shows
- **Source:** Kaggle
- **Features Include:**

  - show_id
  - type (Movie / TV Show)
  - title
  - director
  - cast
  - country
  - date_added
  - release_year
  - rating
  - duration
  - listed_in (genre)
  - description

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

1. Data loading and inspection
2. Handling missing values
3. Distribution analysis using histograms
4. Analysis of categorical features using count plots
5. Box plots to identify outliers
6. Correlation analysis (where applicable)
7. Visualization-based insights

- Movies are more common than TV Shows on Netflix
- Majority of content is released after 2010
- Most popular ratings include TV-MA and TV-14
- The USA has the highest content contribution
